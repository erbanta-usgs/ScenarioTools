As of Feb. 23, 2011, SVN repository contains the following projects and files:

o ScenarioAnalyzer - This is a C# project.  Compile it with MS Visual Studio 2008.  The VS solution file is ScenarioAnalyzer\source\ScenarioManagerMG5.sln

o ScenarioManager - This is a C++ project.  Compile it with Qt Creator.  The project file is ScenarioManager\ScenarioManager.pro.  QtCreator (LGPL license) is available from http://qt.nokia.com/products/developer-tools/.

o Florida_tools_tasks.tsk - This is a Task Coach file containing to-do tasks for the Florida-tools project.  Task Coach is available from SourceForge.net.

o Florida_tools_log.txt - Log of project actions.
