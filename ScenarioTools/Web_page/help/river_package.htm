River Package

 Top  Previous  Next

The River Package simulates flow between the groundwater system and a river, stream, or canal external to the groundwater system

 

Shapefile contents: Polylines representing segments of rivers, streams, or canals.

 

Time-series file contents:  Time-varying stage for each segment.

 

Package-specific settings and properties:

 

•Secondary time-series file: Allows user to activate and deactivate river features during simulation
•Hydraulic conductivity: Hydraulic conductivity of the riverbed material
•River width: Width of the riverbed
•Riverbed thickness: Thickness of the riverbed
•Riverbed bottom elevation: Elevation of the base of the riverbed
