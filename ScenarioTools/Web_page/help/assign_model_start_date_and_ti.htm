Assign model start date and time

 Top  Previous  Next

To assign a start time for a Scenario Manager project, click on the Project menu and select “Simulation Start Time.”  In the “Simulation Start Date and Time” dialog, assign the start year (you can highlight the default year and type the new one). The start time defaults to 00:00:00. Click OK in the “Simulation Start Date and Time” dialog to save the start date and close the dialog.
