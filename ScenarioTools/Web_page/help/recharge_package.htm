Recharge Package

 Top  Previous  Next

The Recharge Package simulates recharge to the groundwater system.

 

Shapefile contents: Polygons representing areas of recharge that are to be controlled by the Recharge Package.

 

Time-series file contents: Time-varying recharge rates.

 

Package-specific settings and properties:

 

NRCHOP: Option that determines which layer receives recharge in each stack of cells or for all cell locations
