Color-fill map layers

 Top  Previous  Next

Each color-fill map layer uses data of a specified category:

 

•Cell-by-cell budget
•Head, drawdown, or concentration
•Data set calculator

 

In a color-fill map layer, the data are symbolized by filling each cell with a color that depends on its position between the minimum and maximum values in the data set. The range of values is represented by a color ramp (a gradational ranges of colors).

 

On the Data Series for a Color-Fill Map Layer dialog, the user can select the end-member colors by clicking the buttons at each end of the color ramp.
