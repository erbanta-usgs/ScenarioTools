Scenario Analyzer example

 Top  Previous 

The Scenario Analyzer project file for this example is SeawatExample.sa, which is located in the examples\SeawatExample folder of the Scenario Tools distribution. The example uses data extracted from SEAWAT output files for the SeawatExample model. Because of the size of these files, they are not distributed with Scenario Tools.

 

In order to use the SeawatExample.sa file, the user will first need to do two things:

 

1.Copy the "examples" folder and all its contents to a folder for which you have write privileges.
2.Run the batch file PrepareExamples.bat in the new location of the "example" folder; this will run the SeawatExample model and generate the required output files. This batch file will take several minutes to run.

 

When Scenario Analyzer opens the SeawatExample.sa file for the first time, loading the file will require several minutes. After the file has been successfully opened, save the file. The data read from the model-output files are saved in the Scenario Analyzer project file. If the data have been saved, the time required for Scenario Analyzer to open the project file the next time will be much shorter.
