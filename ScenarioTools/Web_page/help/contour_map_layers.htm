Contour map layers

 Top  Previous  Next

Each contour map layer uses data of a specified category:

 

•Cell-by-cell budget
•Head, drawdown, or concentration
•Data set calculator

 

In a contour map layer, data are symbolized by lines of equal value (or "contours"). The user has a choice of colors for the lines. The user also specifies the method for defining contour values:

 

•Automatic: Scenario Analyzer selects an appropriate contour interval.

 

•Equal Interval: The user selects a starting contour value and a contour interval.

 

•List of Values: The user provides a comma-separated list of values. Contours representing the provided values are drawn.
