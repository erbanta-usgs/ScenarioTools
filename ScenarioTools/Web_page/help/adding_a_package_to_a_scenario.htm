Adding a package to a scenario

 Top  Previous  Next

To add a package to a scenario, right-click the scenario node in the left panel. In the pop-up menu, select Add New Package and in the submenu select the type of package you want to add. The package types supported in Scenario Manager include:

 

CHD Package (time-variant specified heads)

GHB Package (general head boundaries)

Recharge Package

River Package

Well Package

 

After adding a GHB, Recharge, River, or Well Package, define the unit number identifying the file to which the package will write cell-by-cell budget data:

 

IGHBCB (GHB Package)

IRCHCB (Recharge Package)

IRIVCB (River Package)

IWELCB (Well Package)

 

The "Choose..." button allows the user to select from appropriate "DATA(BINARY)" files listed in the MODFLOW name file. Do not select a file used for hydraulic heads or drawdowns.
