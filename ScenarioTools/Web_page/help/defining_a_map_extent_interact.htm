Defining a map extent interactively

 Top  Previous  Next

To define a map extent for a map interactively, use the Map Designer dialog. To open the Map Designer dialog, in the left panel of the main Scenario Manager window, right-click the map of interest and select Properties. Alternatively, select the map of interest and use the Edit menu and select Element Properties.

 

In the Map Designer dialog, select the Layout tab. With the mouse, click and drag to define the corners of a rectangle enclosing the new extent on the map image. The Save Extent button will be enabled--click it.

 

In the Define Extent dialog that pops up, provide a name for the new extent.
