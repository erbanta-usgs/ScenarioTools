Exporting and running a scenario

 Top  Previous  Next

Select a scenario node in the left panel of the main Scenario Manager window.

 

To export model-input files without running the scenario simulation, use the Export menu and select Export Scenario.

 

After a scenario is exported, the scenario simulation can be run by executing the batch that is generated as part of the export procedure. The batch file will be given a name similar to Run_Scenario_1.bat.

 

To export the model-input files and run the scenario simulation in one step, use the Export menu and select Export Scenario and Run Simulation.
