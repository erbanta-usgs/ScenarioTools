Define the time-series data file for a feature set

 Top  Previous  Next

The time-series data file contains the stress rate or other time-varying property for the features in the feature set.

 

Either double-click on the feature set node in the left panel of the Scenario Manager window or right-click that node, and on the pop-up menu select Edit. This will open a feature-set dialog designed for a feature set of a type corresponding to the parent package. Use the browse button to find a time-series data file containing stress or other time-varying property values for the features.
