Simulated hydraulic head, drawdown or concentration

 Top  Previous  Next

MODFLOW and MODFLOW-related programs are capable of generating binary output files in which simulation results are saved in two-dimensional data arrays by model layer. Each data array stores hydraulic head, drawdown, or solute concentration for all model cells in one layer.

 

An output file of head, drawdown, or concentration can contain data arrays recorded as frequently as every model time step. The modeler uses the Output Control file of MODFLOW (Harbaugh, 2005) to specify the data arrays to be saved and frequency of output.
