Representation of inactive areas

 Top  Previous  Next

In some mapping situations, the user may want the map to show multiple color-fill map layers. To make lower layers visible, it is necessary to make inactive parts of upper layers blank.

 

To specify how areas of inactive cells are represented on maps, use the Project menu and select Settings to open the Scenario Analyzer Project Settings dialog. Select the Inactive Areas tab. The choices and their meanings are:

 

•Leave as is: Inactive areas are not blanked.

 

•Blank by specified IBOUND layer: If a cell is identified as inactive in the selected layer, it is blanked.

 

•Blank where IBOUND inactive in any layer: If a cell is identified as inactive in any layer, it is blanked.

 

•Blank where IBOUND inactive in all layers: If a cell is identified as inactive in all layers, it is blanked.
