The Scenario Manager project file for this example is 500m_example.smgx, which is located in the "examples" folder of the Scenario Tools distribution. This example uses files in a number of the folders under the "examples" folder.

 

To modify the project file or run scenarios, you will first need to copy the "examples" folder and all its contents to a folder for which you have write privileges.

 

The following image shows the main Scenario Manager window with a project containing a number of scenarios.  One scenario has been expanded to show all its scenario elements.

 

 

This project can be opened with Scenario Manager to see the contents of the various elements, and Scenario Manager can be used to run the scenarios or design new ones.
