Shapefile

 Top  Previous  Next

Shapefiles (Environmental Systems Research Institute, Inc., 1998) are used to provide georeferenced information specifying the finite-difference grid used by the model and describing physical features or areas of hydrologic significance to be simulated. A given shapefile can contain geometric features: either points, polylines, or polygons.

 

Users need to ensure that all shapefiles and georeferenced image file, if used, share a common projected coordinate system. If a conflict or apparent conflict is detected, a warning is issued.
