Making maps, charts, and tables

 Top  Previous  Next

Each Scenario Analyzer project can include any number display elements, which include maps, charts, and tables.

 

There are two ways to add a map, chart, or table to a project:

 

•Left-click in the left panel of the main Scenario Analyzer window and select Add New Map, Add New Chart, or Add New Table; or
•Use the Edit menu and select Add New Map, Add New Chart, or Add New Table

 

After adding a display element to a project, data series are added to provide content. Each type of display element has its own methods for adding and manipulating data series:

 

•Making a map
•Making a chart or table
