Data categories

 Top  Previous  Next

Supported data categories for maps include:

 

•Cell-By-Cell Budget for Layer
•Head, Drawdown, or Concentration for Layer
•Data Set Calculator

 

Supported data categories for charts and tables include:

 

•Head, Drawdown, or Concentration at Point
•Cell-By-Cell Budget at Point
•Cell-By-Cell Budget by Cell Group
•Observed Series (SMP file)
•Data Set Calculator
