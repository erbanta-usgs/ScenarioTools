Scenario Tools

 Top  Next

"Scenario Tools" is a suite of two graphical user interface applications designed to facilitate management and analysis of scenarios derived from a base groundwater model that uses one of several versions of MODFLOW for simulation. The suite includes the following applications:

 

•Scenario Manager, which allows the user to design and run scenarios; and
•Scenario Analyzer, which enables the user to easily create maps, charts, and tables based on output generated by MODFLOW models.

 

A Glossary defines a number of terms used in the help file.

 

The Examples section provides brief examples of each application.

 

This help file contains much of the information a user would need to use Scenario Manager and Scenario Tools; however, not all topics are covered in detail in this file. For details on  aspects of these applications not covered in the help file, please see Banta (2014).
