Cell-by-cell budget

 Top  Previous  Next

MODFLOW and MODFLOW-related programs are capable of generating binary output files in which simulated cell-by-cell budget components are saved.

 

The data contained in the cell-by-cell budget file represent components of flow simulated by the MODFLOW packages that are active during a simulation.

 

An output file cell-by-cell budget components can contain data recorded as frequently as every model time step. The modeler uses input for each  package and the Output Control file of MODFLOW (Harbaugh, 2005) to specify the data to be saved and frequency of output. The Scenario Analyzer user generally chooses among a variety of data identifiers when selecting a data set from a cell-by-cell budget file, where each identifier is associated with a particular package and flow component. See Banta (2014) for a list of data identifiers and their meanings.
