HNOFLO

 Top  Previous  Next

HNOFLO is the value of head assigned to all model cells defined as no-flow. Scenario Analyzer needs this value (as it is assigned in the MODFLOW Basic Package input file) to correctly interpret no-flow cells in arrays of hydraulic head or drawdown generated by MODFLOW.

 

To assign this value, use the Project menu and select Settings to open the Scenario Analyzer Project Settings dialog. Then provide the value in the HNOFLO field.
