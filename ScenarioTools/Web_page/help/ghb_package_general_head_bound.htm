GHB Package (general head boundaries)

 Top  Previous  Next

The GHB Package simulates general head boundaries. A general head boundary is a head-dependent source or sink of water external to the simulated groundwater system.

 

Shapefile contents: Polygons representing areas of water sources or sinks external to the groundwater system that are to be controlled by the GHB Package.

 

Time-series file contents: Time-varying specified hydraulic head (stage) for the external sources or sinks.

 

Package-specific settings and properties:

 

•Secondary time-series file: Allows user to activate and deactivate GHB features during simulation
•Leakance of GHB features
•Layer in which GHB cells in a feature set are to be simulated
