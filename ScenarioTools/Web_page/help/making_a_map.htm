Making a map

 Top  Previous  Next

When a project contains one or more maps, a model-grid shapefile needs to be defined for the project.

 

There are two ways to add a map to a project:

 

•Left-click in the left panel of the main Scenario Analyzer window and select Add New Map, or
•Use the Edit menu and select Add New Map

 

After a map has been added to a project, content can be added to the map in the form of one or more data series. For a map, each data series generates either a contour map layer or a color-fill map layer.

 

A data series for a map can be generated from any of three data sources:

 

•A binary MODFLOW output file of hydraulic head, drawdown, or solute concentration;
•A binary MODFLOW output file of cell-by-cell budget components; or
•A data set calculator data series

 

To add a map layer to a map, select the map in the left panel of the main Scenario Analyzer window, and then either:

 

•Use the Edit menu, or
•Right-click the map symbol or map name in the left panel

 

Either action will provide the following options:

 

•Add New Contour Layer
•Add New Color-Fill Layer

 

Select one of these choices to add the desired type of map layer.
