Adding a new scenario to the project

 Top  Previous  Next

Use any of the three ways to add a new scenario to a Scenario Manager project: (1) On the Edit menu, select “Add New Scenario,” (2) Right-click in the left panel and select “Add Scenario” from the pop-up menu, or (3) Press <Ctrl>-N on the keyboard. To provide a name of your choice for the scenario, in the “Scenario ID” dialog that appears, change the Scenario ID and click OK. You can also rename the scenario after you have created the scenario by right-clicking on the scenario node in the left panel of the main Scenario Manager window and selecting Rename.
