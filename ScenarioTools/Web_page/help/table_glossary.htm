In Scenario Analyzer, a table can display one or more data series where each row  of the table represents a unique time. For a table, a data series is a time series of data values.

 

Each data series belonging to a table can be represented on the table as a column. Alternatively, visibility of a data series can be turned off.

 

The rows of the table are determined by the entries in the first data series defined for the table. As a result, the table will contain a row representing the time of each member of the first data series. Values for the remaining columns are linearly interpolated from the corresponding data series as needed.
