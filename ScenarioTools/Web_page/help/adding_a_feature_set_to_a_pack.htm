Adding a feature set to a package

 Top  Previous  Next

Any package can include any number of feature sets. To add a feature set to a package, right-click the package node in the left panel of the main Scenario Manager window and select "Add New Feature Set."

 

Each feature set has an associated shapefile, which contains features to be simulated; an associated time-series data file, which contains time-varying stresses or properties related to the features in a feature set; and other data as needed by the package to which the feature set belongs.

 

After a feature set has been added, the following steps need to be completed to have the feature set be represented in simulation based on a scenario:

 

•Define the shapefile
•Define the time-series data file
•Assign attribute linking the shapefile to the time-series data file
•Define package-specific settings and properties for the feature set
