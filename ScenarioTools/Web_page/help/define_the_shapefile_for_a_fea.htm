Define the shapefile for a feature set

 Top  Previous  Next

Either double-click on the feature set node in the left panel of the Scenario Manager window or right-click that node, and on the pop-up menu select Edit. This will open a feature-set dialog designed for a feature set of a type corresponding to the parent package. Use the browse button to find a shapefile containing features appropriate for the parent package, and assign that file to the Shapefile field.
