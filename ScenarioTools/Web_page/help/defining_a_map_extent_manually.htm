Defining a map extent manually

 Top  Previous  Next

To define a map extent for a map manually, use the Map Designer dialog. To open the Map Designer dialog, in the left panel of the main Scenario Manager window, right-click the map of interest and select Properties. Alternatively, select the map of interest and use the Edit menu and select Element Properties.

 

In the Map Designer dialog, select the Extent tab. Select the Manual option.

 

The coordinates defining the limits of the current extent can be edited. This is a convenient way to fine-tune the extent.

 

To define a new extent, replace the name of the current extent with a new name, which will identify the new extent. Then provide new coordinates for the limits of the new extent.
