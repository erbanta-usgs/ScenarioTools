Background image

 Top  Previous  Next

A Scenario Analyzer project can have a background image, which can be used as a background for any map in the project. The background image must be available as a georeferenced image file.

 

To assign a background image for a Scenario Analyzer project, open the Scenario Analyzer Project Settings dialog (use the Project menu and select Settings). Use the browse button for the "Georeferenced Background Image File" field and select a GeoTIFF file or other file supported by the Geospatial Data Abstraction Library (GDAL) (http://www.gdal.org/).

 

Users need to ensure that the model-grid shapefile and the georeferenced image file, if used, share a common projected coordinate system. If a conflict or apparent conflict is detected, a warning is issued.
