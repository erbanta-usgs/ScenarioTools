Map layers

 Top  Previous  Next

Any map can contain one or more map layers. Each map layer is either a contour map layer or a color-fill map layer and is based on a data series.

 

To add a map layer, first select the desired map node in the left panel of the main Scenario Analyzer window. Either right-click the node or use the Edit menu. Then, select one of the options:

 

•Add New Contour Layer
•Add New Color-Fill Layer
