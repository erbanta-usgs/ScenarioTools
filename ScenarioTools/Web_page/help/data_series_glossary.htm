Data series

 Top  Previous  Next

In Scenario Analyzer, a data series is a source of data for a map, chart, or table. The data series includes settings and information required to obtain or generate the data. Each map, chart, or table can include one or more data series.

 

A data series for a chart or table can be obtained from, or generated by, a number of data sources, including:

 

•Values from binary MODFLOW output files of simulated hydraulic head, drawdown, or solute concentration.
•Values from binary MODFLOW output files of simulated cell-by-cell budget components.
•Values from a time-series data file
•A data series also can be defined as the result of calculations based on one or more other data series included in the chart. The data set calculator is used to define this type of data series.

 

Each data series includes a data category and a data set.
