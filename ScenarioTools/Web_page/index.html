Scenario Tools: Scenario Manager and Scenario Analyzer

Graphical user interfaces for managing and analyzing MODFLOW groundwater-model scenarios

Information   Screenshots   Documentation   Download   Support   Notification of releases   Related programs

Information

Scenario Tools is a suite of two graphical user interface (GUI) applications designed for situations where a user wants to:
(1) Create model scenarios derived from a base model that uses MODFLOW or a related groundwater-modeling program; and
(2) Analyze results generated by running such a model scenario.

The applications support any MODFLOW-based model that generates binary files of hydraulic head, drawdown, solute concentration, or cell-by-cell budget in the forms generated by MODFLOW–2005, SEAWAT, or MT3DMS.

The applications are:

  • Scenario Manager, which facilitates the design of scenarios derived from a base model; and
  • Scenario Analyzer, which is designed to create maps, charts, and tables using data extracted from binary output files of hydraulic head, drawdown, solute concentration, and cell-by-cell budget terms.
